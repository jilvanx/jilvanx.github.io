## Install 🚀
* Clone the project
```sh
git clone https://github.com/jilvanx/jilvanx.github.io.git
```

* Install the dependencies and run.
```sh
$ cd jilvanx.github.io
$ yarn install
$ yarn start
```
Or
```sh
$ cd jilvanx.github.io
$ npm install
$ npm run start
```
